# E04-T01 — Row selection

Status: `pending`

## User outcome

Users can confidently choose the exact issues they want to change.

## Acceptance criteria

- Rows expose accessible checkboxes and a visible selected count.
- Select-all selects only the issues visible in the current filtered view.
- Shift-click selects a contiguous range.
- Keyboard users can extend, reduce, and clear selection.
- Sorting preserves selection while view or workspace changes remove invalid rows.
