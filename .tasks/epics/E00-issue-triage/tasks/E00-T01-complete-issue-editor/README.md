# E00-T01 — Complete issue editor

Status: `done`

## User outcome

Users can create an issue with meaningful metadata and later edit its title,
description, status, priority, project, assignee, and labels in one place.

## Acceptance criteria

- Create supports status, priority, project, assignee, and labels.
- Detail supports editing title, description, project, assignee, and labels.
- Save, pending, success, and failure states are visible.
- Delete requires explicit confirmation.
- Mutations update every issue view.
