# E00-T02 — Filter and sort

Status: `done`

## User outcome

Users can reduce a list by status, priority, assignee, and label and can order it
by recently updated, priority, or identifier.

## Acceptance criteria

- Filters compose and can be cleared in one action.
- The toolbar shows when filters are active.
- Sorting works consistently across active, team, and project views.
- Empty filtered results explain how to recover.
