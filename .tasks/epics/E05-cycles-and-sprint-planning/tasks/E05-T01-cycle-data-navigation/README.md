# E05-T01 — Cycle data and navigation

Status: `pending`

## User outcome

Users can find the right current, upcoming, or recent cycle for each team.

## Acceptance criteria

- The API exposes cycle name, number, team, dates, and lifecycle status.
- Team navigation surfaces current, upcoming, and recent cycles.
- Cycle routes are workspace- and team-scoped.
- Refresh and browser navigation restore the same cycle.
- Loading, empty, permission, and unavailable states are understandable.
