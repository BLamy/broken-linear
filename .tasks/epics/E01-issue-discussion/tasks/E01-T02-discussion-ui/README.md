# E01-T02 — Discussion UI

Status: `done`

## User outcome

The issue detail view contains a readable, useful conversation thread.

## Acceptance criteria

- Each comment shows its author, avatar or initials, timestamp, and body.
- An empty issue explains that the conversation has not started.
- The composer posts a comment without closing the issue.
- A user can enter edit mode and save or cancel.
- Delete requires confirmation.
